# Electronic Arts (electronic-arts)
Electronic Arts is a Fortune 500 company that provides developer APIs and integration capabilities through its developer platform. Electronic Arts provides developer APIs for game services, player authentication, leaderboards, and social features. The EA platform enables game developers to integrate EA's gaming services.
